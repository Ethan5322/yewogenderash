03 — Landing

Homepage · /

Deliberately not a marketing splash — it opens straight into live campaigns, then explains why the platform is trustworthy. No hero banner, no owner sign-up shouting.

▦
▦
▦

Section 1 — “Active campaigns”

A heading + one-line reassurance, a View all campaigns → button, then a responsive grid of up to 9 campaign cards (1 / 2 / 3 columns by screen size). Empty state: a friendly “check back soon” card.

Section 2 — “Built for trust, end to end”

On a tinted band: four icon cards — Verified owners, One campaign, one code, Separated ledgers, Instant confirmation. Purely informational, no CTA.

Tone: calm, credible, donor-first. The “sell” is trust, not hype.

07 — Become a fundraiser & sign-in

Onboarding wizard & auth

How a new fundraiser is verified, and how people sign in. Public-facing, but gated and serious.

Get started / KYC wizard · /start → /start/…

1 · Terms 2 · Documents 3 · Verify (live face) 4 · Review
  • Step-by-step, progress-tracked flow: accept terms & fees → upload ID + supporting docs → live biometric selfie (with a phone-capture handoff via QR) → review & submit.
  • On submit the applicant becomes Pending and enters the admin KYC queue.
  • Mobile capture · /capture/[token] — a phone-only page to take the live selfie, linked to the desktop session in real time.

Authentication

PagePurpose
/loginStandard email + password sign-in (donors / owners).
/registerCreate an account (name, email, phone, password + email OTP).
/fundraiser-loginSign in with the printed fundraiser code + password (+ biometric where enrolled).

Auth pages are centered single cards on a soft background, logo at the top — calm and focused.
